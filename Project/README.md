# TripMate Android App

This repository now contains an Android Jetpack Compose project.

## Requirements

1. Android Studio (Koala or newer recommended)
2. Android SDK 34
3. JDK 17

## Run In Android Studio

1. Open this folder in Android Studio.
2. Wait for Gradle sync to complete.
3. Select an emulator or physical device.
4. Click Run for the app module.

## Run From Terminal (Windows)

1. Ensure Gradle wrapper exists (`gradlew.bat`).
2. Build:
   `gradlew.bat assembleDebug`
3. Install to a connected device:
   `gradlew.bat installDebug`
